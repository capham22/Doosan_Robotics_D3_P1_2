# 🤖 Intelligent Robotics Project Collection

다양한 목적의 **지능형 로봇 시스템**을 모아놓은 종합 프로젝트 공간입니다.  

## 📁 프로젝트 목록

| 프로젝트명 | 설명 | 주요 기술 |
|------------|------|-----------|
| 🚗 **도로지킴이** | 불법 주정차 차량을 탐지하고 자동으로 대응하는 자율 로봇 시스템 | YOLO, ROS 2, TF2, Nav2 |
| 🦾 **요양보조로봇** | 요양 업무 종사자 협동 로봇 팔 | YOLO, ROS 2, TF2, Doosanrobot api |

| 분류        | 기술 구성                        |
|-------------|----------------------------------|
